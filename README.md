# SwiftfulThinkingBootcamp

A comprehensive SwiftUI learning project covering fundamental to advanced concepts through practical examples and bootcamp-style tutorials.

## Overview

This project contains a collection of SwiftUI bootcamp files that demonstrate various SwiftUI concepts and components. Each file focuses on a specific SwiftUI feature or concept, making it easy to learn and reference different aspects of SwiftUI development.

## Project Structure

The project is organized with individual bootcamp files, each focusing on a specific SwiftUI concept:

- **Basic UI Elements**: Text, Shapes, Colors, Gradients, Icons, Images
- **Layout & Styling**: Frame, Background/Overlay, Stacks, Padding, Spacer
- **Components**: Buttons, Lists, Navigation, Tab Views, Sheets
- **State Management**: State, Binding, Conditional Views, Animations
- **User Input**: TextFields, Pickers, Sliders, Toggles, Date Pickers
- **Advanced Features**: ScrollView, Grid, SafeArea, Dark Mode

## Getting Started

1. Clone or download the repository
2. Open `SwiftfulThinkingBootcamp.xcodeproj` in Xcode
3. Build and run the project
4. Explore different bootcamp files to learn various SwiftUI concepts

## Requirements

- iOS 14.0+
- Xcode 12.0+
- Swift 5.0+

## Learning Path

The bootcamp files are organized to provide a progressive learning experience:

1. Start with basic UI elements (Text, Shapes, Colors)
2. Move to layout concepts (Stacks, Frame, Padding)
3. Learn about state management (State, Binding)
4. Explore user interface components (Buttons, Lists, Navigation)
5. Advance to animations and transitions
6. Understand platform-specific features (Dark Mode, SafeArea)
